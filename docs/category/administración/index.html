Administración

En el Menú de Administración, te encuentras con una variedad de funciones cruciales para la gestión académica. Puedes realizar acciones como abrir o cerrar evaluaciones, trabajar con cargas académicas (crear nuevas, ver detalles, asignar docentes, entre otros), gestionar estudiantes, y registrar evaluaciones manualmente. Además, cuentas con la posibilidad de administrar grupos, ya sea creando, editando o eliminando. Este menú ofrece herramientas esenciales para un control académico efectivo.

📄️ Abrir/Cerrar Evaluaciones

Esta opción te brinda un control preciso sobre el acceso a las evaluaciones. Puedes abrir o cerrar evaluaciones según las necesidades del proceso académico, permitiéndote gestionar de manera efectiva el tiempo de disponibilidad de cada evaluación. Con esta funcionalidad, aseguras un mayor control y planificación en la administración de evaluaciones.

📄️ Cargas Académicas

En este menú, puedes crear nuevas cargas académicas, ajustar o eliminar las existentes y definir su estado según el ciclo académico. Además, tienes la capacidad de gestionar estudiantes, asignándolos o cambiándolos de carga académica, así como también realizar las mismas acciones con el personal docente. También ofrece la posibilidad de realizar el registro de evaluaciones y llevar un control detallado de la asistencia de cada Carga Académica.

📄️ Registro de Evaluaciones

La opción de "Registro de Evaluaciones" te brinda la capacidad de realizar un seguimiento detallado del desempeño académico de los estudiantes. Puedes asignar calificaciones a cada estudiante en diversos tipos de evaluaciones. Además, esta función facilita la generación de reportes en formato Excel, permitiéndote analizar y compartir fácilmente la información relacionada con las evaluaciones.

📄️ Grupos

En la opción de Grupos puedes crear nuevos grupos, personalizar los existentes o eliminar aquellos que ya no son necesarios. Además, esta opción te permite gestionar la asignación de estudiantes a cada grupo. Para un seguimiento eficiente, puedes generar reportes de cada grupo.
